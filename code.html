Problem statement :


Write a code in any programming language that counts the digits in a number.

Program:

#include < bits/stdc++.h >
using namespace std;
int main()
{
     int x;int cnt=0;
     cin>>x;
     if(x==0)
     return 0;
while(x)
{
     cnt++;
     x=x/10;
}
cout<<Number of digits are:<<cnt<<endl;
return 0;
}


Input:123456

Output:6

Variables:x,cnt

Complexity:O(n)
